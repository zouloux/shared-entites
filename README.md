# Shared Entities


## Install

`npm i @zouloux/shared-entities`

## Usage

##### Main module
- `import { ... } from "@zouloux/shared-entities"`

##### Sub module
- `import { ... } from "@zouloux/shared-entities/submodule"`

## Build commands

##### Build
- `npm run build`
##### Test
- `npm run test`
##### Publish
- `npm run publish`
